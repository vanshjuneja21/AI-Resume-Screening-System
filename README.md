# AI Resume Screening System

## Overview
This project is an NLP-based resume screening system that matches resumes with job descriptions using TF-IDF vectorization and cosine similarity.

## Features
- NLP preprocessing (tokenization, stopword removal)
- TF-IDF vectorization
- Cosine similarity scoring
- Resume ranking system
- Category-based filtering

## Tech Stack
- Python
- Pandas
- NLTK
- Scikit-learn

## How It Works
1. Preprocess resumes using NLP techniques  
2. Convert text into numerical vectors using TF-IDF  
3. Compute similarity with job description  
4. Rank resumes based on relevance score  

## Dataset
Dataset used: Resume Dataset (Category + Resume text)

## How to Run
1. Open the notebook in Google Colab  
2. Upload dataset  
3. Run all cells  

## Output
- Ranked resumes based on similarity score  
- Top candidates shortlisted automatically
